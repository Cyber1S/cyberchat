# Buttons

Buttons can be used by `c1chat-button` component in `vue`. You can play with all the props here ðŸ‘‰ [c1chat-button](/story/components-button--primary). The button component is made on top of [foundation button](https://get.foundation/sites/docs/button.html)
so the style of button can be used outside of vue with `button` class.

Other than declred props the `c1chat-button` can be customised by providing custom classnames through `class-names` prop.

<Story name="default">
  {{
    template: `<c1chat-button size="" title="Press me">Default</c1chat-button>`,
  }}
</Story>

- While naming button labels make sure to describe the action with some context.
- For HTML forms use `<c1chat-button type="submit">` instead of `<input type="submit" />`
- Provide title or tooltip if possible.

## Button sizes

Size of the buttons can be set with `size` to `c1chat-button`. By default medium is the default size of a button. Other sizes are `tiny, small, medium, large`.

<Canvas>
  <Story name="sizes">
    {{
      template: `<div class="row">
          <div class="margin-horizontal-2">
              <c1chat-button size="tiny">Tiny</c1chat-button>
          </div>
          <div class="margin-horizontal-2">
              <c1chat-button size="small">Small</c1chat-button>
          </div>
          <div class="margin-horizontal-2">
              <c1chat-button size="">Default</c1chat-button>
          </div>
          <div class="margin-horizontal-2">
              <c1chat-button size="large">Large</c1chat-button>
          </div>
        </div>`,
    }}
  </Story>
</Canvas>

Buttons can expanded to full width by setting the `is-expanded` prop. For with by screen size or to make it responsive try with the classnames on
[Responsive Expanded buttons](https://get.foundation/sites/docs/button.html#responsive-expanded-buttons) with foundation.

<Canvas>
  <Story name="expanded">
    {{
      template: `<div class="row">
          <div class="columns">
              <c1chat-button size="large" is-expanded title="Micheal Scott said the rest">This is buttons is very long</c1chat-button>
          </div>
        </div>`,
    }}
  </Story>
</Canvas>

## Button colors

Cyber1SChat comes with predefined set of colors for buttons. The color of the button can be set with `color-scheme` to `c1chat-button`.
Default color scheme is `primary`, other color-schemes are `secondary, success, alert, warning`.

</Canvas>

## Button style variations

There are diffrent styles that comes with cyber1schat buttons. This can set with `variants` prop for `c1chat-button`. By deault the filled will be the style, other variations are `smooth, hollow, clear, link`

- Primary form actions like Save details, Create campaign etc should use default style.
- The Cancel action has to use the smooth style.

<Canvas>
  <Story name="variants">
    {{
      template: `<div class="row">
          <div class="margin-horizontal-2">
              <c1chat-button >Default</c1chat-button>
          </div>
          <div class="margin-horizontal-2">
              <c1chat-button variant="smooth" color-scheme="alert">Smooth</c1chat-button>
          </div>
          <div class="margin-horizontal-2">
              <c1chat-button color-scheme="secondary" variant="hollow">Hollow</c1chat-button>
          </div>
          <div class="margin-horizontal-2">
              <c1chat-button color-scheme="success" variant="clear">Success</c1chat-button>
          </div>
          <div class="margin-horizontal-2">
              <c1chat-button color-scheme="alert" variant="link">Link</c1chat-button>
          </div>
        </div>`,
    }}
  </Story>
</Canvas>

## Button states

The button can be disbaled with `is-disabled` which accepts a boolean value. The button can set to a loading state with `is-loading` which accepts a boolean value.

These are states other than the default interaction states like hover, active, focused.

<Canvas>
  <Story name="states">
    {{
      template: `<div class="row">
          <div class="margin-horizontal-2">
              <c1chat-button color-scheme="alert" is-disabled>disabled :/</c1chat-button>
          </div>
          <div class="margin-horizontal-2">
              <c1chat-button variant="smooth" color-scheme="primary" is-loading>Loading...</c1chat-button>
          </div>
        </div>`,
    }}
  </Story>
</Canvas>

## Button with icon and text

Cyber1SChat currently use [IonIcons version 2](https://ionic.io/ionicons/v2) which are font-icons. The button position is fixed on the left.
The prop `icon` can be used to pass the **ion icon classname** to render the icon.

<Canvas>
  <Story name="icons">
    {{
      template: `<div class="row">
          <div class="margin-horizontal-2">
              <c1chat-button color-scheme="alert" icon="ion-stop">Stop</c1chat-button>
          </div>
          <div class="margin-horizontal-2">
              <c1chat-button icon="ion-person" variant="hollow">Hollow man</c1chat-button>
          </div>
        </div>`,
    }}
  </Story>
</Canvas>
